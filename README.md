## Features

- Resume upload form
- PDF text extraction
- Gemini AI candidate analysis
- Structured output parsing
- Candidate score threshold routing
- Google Sheets integration
- Recruiter email notification

---

## Tech Stack

- n8n
- Gemini AI
- Gmail API
- Google Sheets API
- PDF Extraction

---

## Workflow

Upload Resume
→ Extract Resume Text
→ Gemini AI Analysis
→ Structured Output
→ Candidate Routing
→ Save to Sheets
→ Recruiter Notification

## Use Cases

- HR automation
- AI recruitment systems
- Resume screening
- Applicant tracking automation
