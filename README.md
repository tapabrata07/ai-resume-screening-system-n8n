# AI Resume Screening System using n8n + Gemini

An AI-powered recruitment automation workflow built using n8n and Google Gemini.

## Workflow Overview

![Workflow Screenshot](screenshots/workflow.png)

---

## Features

- Resume upload form
- PDF text extraction
- Gemini AI candidate analysis
- Structured output parsing
- Candidate score threshold routing
- Google Sheets integration
- Recruiter email notification

---

## Tech Stack

- n8n
- Gemini AI
- Gmail API
- Google Sheets API
- PDF Extraction

---

## Workflow

Upload Resume
→ Extract Resume Text
→ Gemini AI Analysis
→ Structured Output
→ Candidate Routing
→ Save to Sheets
→ Recruiter Notification

---

## Use Cases

- HR automation
- AI recruitment systems
- Resume screening
- Applicant tracking automation

## Setup

1. Import workflow JSON
2. Connect Gemini API
3. Connect Google Sheets
4. Connect Gmail
